# Retro Terminal Simulator

A retro-inspired terminal simulator built using HTML, CSS, and JavaScript.

## Run it
Open `index.html` in any modern browser.

## Features
- Retro green terminal theme
- Typing animation
- Custom terminal commands
- Command history
- Matrix mode
- Clear screen
- Help menu
- Responsive design

## Commands

| Command | Description |
|----------|-------------|
| help | Show all commands |
| about | About the project |
| date | Current date and time |
| clear | Clear terminal |
| matrix | Toggle Matrix mode |
| joke | Display random joke |
| echo [text] | Print custom text |
| whoami | Display user |
| projects | Show project list |

## Technologies Used
- HTML5
- CSS3
- JavaScript
