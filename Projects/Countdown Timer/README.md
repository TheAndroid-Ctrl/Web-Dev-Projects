# ⏳ Countdown Timer

A customizable countdown timer with alarm sound, pause/resume controls, and a visual circular progress ring.

## Features
- ⏱️ Set minutes and seconds
- ▶️ Start / ⏸️ Pause / 🔄 Reset controls
- 🔊 Alarm sound when time is up
- 🎨 Circular progress ring animation
- 📱 Responsive, attractive UI

## Usage
Open `index.html` in your browser.  
Enter time, start the countdown, and watch the progress ring animate.
